# 产品概览

TechBao Data Platform 是一个以开源组件集成为核心的数据平台方案。它面向需要快速搭建数据管理、数据开发、数据仓库、BI 分析和数据治理能力的团队。

## 用户角色

- 平台管理员：负责组件部署、认证、权限、配置和运维。
- 数据开发人员：负责数据同步、清洗、建模和调度任务。
- 数据分析人员：负责仪表盘、报表和自助分析。
- 数据治理人员：负责数据目录、元数据、血缘、标签和责任人。
- 应用开发人员：通过 API 网关和 Directus 使用业务数据服务。

## 核心能力

- 通过 Directus 快速建立业务数据管理界面和自动 API。
- 通过 PostgreSQL 承载业务数据和平台配置。
- 通过 Doris 提供高性能分析型查询。
- 通过 DolphinScheduler 统一编排数据任务。
- 通过 Superset 提供 BI 和可视化分析。
- 通过 DataHub 建立数据资产目录。
- 通过 APISIX 统一管理 API 出口。
- 通过 Keycloak 提供统一认证和 SSO。
- 通过 MinIO 提供对象存储能力。

## 非目标

- 当前不开发定制业务系统。
- 当前不内置行业数据模型。
- 当前不替代各开源组件自身的管理能力。
- 当前不承诺所有组件已完成生产级集成。
