# ADR 0001：开源组件集成优先

## 状态

已接受

## 背景

数据平台涉及数据管理、数据开发、数据仓库、数据目录、BI、API 网关、统一认证和对象存储等多类能力。如果全部自研，将带来较高的研发、测试、运维和安全成本。

## 决策

TechBao Data Platform 采用“成熟开源组件优先、系统集成为主、尽可能少开发”的建设策略。

## 影响

- 平台工作重点转向组件选型、部署编排、认证集成、数据链路、元数据治理和运维规范。
- 定制开发只用于必要的适配层、自动化脚本和补充工具。
- 新增组件或替换组件时，需要补充选型说明和影响评估。

## 备选方案

- 全部自研：控制力强，但成本高、周期长、风险大。
- 采购商业平台：交付快，但成本、锁定风险和定制边界需要评估。
- 单组件平台化：起步简单，但难以覆盖完整数据平台能力。
